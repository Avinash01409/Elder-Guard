# Elder-Guard
Elder Guard – Community-Based Scam Alert App  Elder Guard is a location-based community safety application designed to help people stay informed about scams happening around them. The app allows users to join communities based on their selected location, report suspicious activities, and receive verified alerts to prevent fraud.  
🚀 Features
Location-based community system
Report scams happening nearby
Admin verification before publishing alerts
Verified badge for trusted scam reports
Real-time notifications for community members
Fake link scanner to detect phishing links
User authentication with email verification
Profile management
🛠 Tech Stack

Frontend: XML
Backend: Kotlin
Database & Authentication: Firebase
Notifications: OneSignal

🔔 How It Works
Users sign up using email verification
Join a community based on selected location
Report scam incidents in the community
Admin verifies the report
Verified scams are published with a verification tag
Community members receive real-time alerts

🎯 Goal

The main goal of Elder Guard is to build a safer community by sharing verified scam alerts and helping users detect phishing links before they become victims.
